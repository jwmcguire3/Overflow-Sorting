# Overflow Sorting

Expo managed React Native project scaffold for the Overflow Sorting puzzle game.

## Prerequisites

- Node.js 20+
- npm 10+

## Setup

```bash
npm install
```

## Run the app (development server)

```bash
npm run dev
```

## Typecheck

```bash
npm run typecheck
```

## Lint

```bash
npm run lint
```

## Tests

```bash
npm run test
```
